【図解】はじめてのクラスとオブジェクト

サイコロのプログラムを例に、プログラミングの重要な考え方を学ぼう！

1. サイコロの状態をデータで表現する

プログラムでは、サイコロの6つの面を1つのリスト faces で管理します。リストの各番号（インデックス）が、サイコロのどの面に対応するか、という「ルール」を決めることが重要です。

展開図とリストの対応

faces[0]
左面
faces[3]
正面
faces[1]
右面
faces[2]
背面
faces[4]
底面
faces[5]

2. 「クラス（設計図）」と「オブジェクト（実体）」

オブジェクト指向とは、関連する「データ」と「操作」をひとまとめにして扱う考え方です。そのために「クラス」と「オブジェクト」を使います。

クラス：設計図

「サイコロとは何か」を定義します。

class Dice:
  # データを持つ場所
  def __init__(self, faces):
    self.faces = faces

  # 操作（メソッド）
  def roll(self, direction):

オブジェクト：実体

設計図から作られた、具体的なサイコロ。

# 6つの面の数字を与えて...
faces1 = [1, 2, 3, 4, 5, 6]
# 1つ目のサイコロを作る！
dice_A = Dice(faces1)

# 別の数字でもう1つ作ることもできる
faces2 = [10, 20, 30, 40, 50, 60]
dice_B = Dice(faces2)
