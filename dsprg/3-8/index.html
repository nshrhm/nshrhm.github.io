3.8.3 欠損値の扱い

  • 表現：RはNA、Pythonはnp.nan
  • 判定：== ではなく専用関数
    • Python：np.isnan()
    • R：is.na()
  • 欠損を無視した計算
    • Python：np.nansum, np.nanmean
    • R：na.rm = TRUE

欠損判定は == ではない（重要）

間違い

x = np.nan
x == np.nan  # False

x <- NA
x == NA  # NA

正しい

  • Python：np.isnan(x)
  • R：is.na(x)

代表サンプル（欠損値の判定と計算）

  • 目的：欠損の判定→カウント→欠損無視計算→除外 を一通りやる
  • ファイル：my3-8-03_missing_value.py / my3-8-03_missing_value.R
  • 実行：python my3-8-03_missing_value.py / Rscript my3-8-03_missing_value.R

確認ポイント

  • np.isnan / is.na の結果
  • np.nanmean と na.rm = TRUE の違い
  • 欠損除外（~np.isnan / !is.na）
import numpy as np
my_data = np.array([10, np.nan, 30, 40, np.nan])

is_missing = np.isnan(my_data)
print(is_missing)
print(np.sum(is_missing))

print(np.mean(my_data))
print(np.nanmean(my_data))

print(my_data[~np.isnan(my_data)])

Copilot活用（欠損値）

  • まずは「このデータで欠損を除外する方法」を聞く
  • 処理方法はデータの性質で最適が変わる
  • 提案は必ず試して、結果が妥当か確認する
