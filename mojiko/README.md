
# 門司港レトロ旅のしおり PWA

小倉南区葉山町発、2026年5月9日（土）9:00出発・16:00帰着の門司港レトロ日帰りドライブ用Webしおりです。

## 開き方

1. ZIPを展開します。
2. `index.html` をブラウザで開きます。
3. PWAとしてService Workerを使う場合は、展開したフォルダで以下を実行し、`http://localhost:8000/` を開いてください。

```bash
python3 -m http.server 8000
```

スマートフォンで使う場合は、同じネットワークからPCのローカルサーバーへアクセスするか、任意の静的ホスティングにアップロードしてください。Googleマップ、外部写真、公式リンクはオンライン接続が必要です。

## 主なページ

- `index.html`: 今日のしおり
- `plans.html`: プランA/B分岐
- `map.html`: Googleマップ活用
- `restaurants.html`: グルメ候補
- `story.html`: 歴史と伝説
- `links.html`: リンク集
