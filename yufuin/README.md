# 湯布院お泊りBBQ 旅のしおり

AMBER Yufuin【Villa1】での家族旅行用に作成した静的Webしおりです。
スマートフォン閲覧、印刷、Googleマップ活用、PWA風表示を想定しています。

## 使い方

1. `index.html` を開くと、旅のしおりが表示されます。
2. `links.html` は有益リンク集です。
3. `print.html` は紙で確認しやすい印刷ページです。
4. スマホで共有する場合は、フォルダごとWebサーバー、GitHub Pages、Netlifyなどに置くと便利です。

## PWAについて

`manifest.webmanifest` と `sw.js` を同梱しています。HTTPSのWebサーバー上で開くと、ホーム画面追加やオフラインキャッシュが使えます。
ZIPを解凍してローカルの `file://` で開いた場合、ブラウザ仕様によりサービスワーカーは登録されません。その場合も、閲覧・印刷・Googleマップリンク・チェックリストは利用できます。

## 計画を変更したい場合

主なデータは `data.js` にまとめています。
日程、宿情報、リンク、買い出しリストを更新する場合は、このファイルを編集してください。
大きな文章やレイアウトは `index.html`、印刷専用の文面は `print.html` を編集します。

## 注意

公開版では予約番号・代表者名などの予約情報を伏せています。実値は家族内共有メモで管理してください。

## 写真について

本文中の写真はWikimedia Commonsの外部画像URLを表示しています。オフライン時は写真が表示されない場合がありますが、ローカルのイラスト・QR・本文は表示されます。
写真クレジットは `links.html` に記載しています。
